Self-Pruning Neural Network (Learned Gating)

This project implements a dynamic model compression technique where a neural network learns to prune its own weights during training using learnable gates and L1 regularization.📌 Project OverviewUnlike static post-training pruning, this architecture uses a Gating Mechanism to identify and remove redundant connections on the fly.Core Features:Custom PrunableLinear Layer: Implements $W_{eff} = W \odot \sigma(gate\_scores)$.Sparsity-Aware Loss: Total Loss = $\mathcal{L}_{task} + \lambda \cdot \mathcal{L}_{sparsity}$.Dynamic Adaptation: The network architecture optimizes itself for the given computational budget ($\lambda$).📂 Structuresrc/model.py: Model architecture and custom prunable layers.src/train.py: Data pipeline, training loop, and evaluation.results/: Saved distribution plots and performance metrics.📊 Performance SummaryLambda (λ)Accuracy (%)Sparsity (%)Status$1 \times 10^{-6}$~55%0%Dense Baseline$5 \times 10^{-3}$~52%~35%Optimal Pruning$1 \times 10^{-2}$~45%~80%Aggressive Compression🛠️ Implementation DetailsGate Transformation: We use a Sigmoid function to squash learnable gate scores into a $[0, 1]$ range.L1 Regularization: Applied to the gates to provide a constant pressure toward zero. This results in a bimodal distribution where weights are either "fully on" or "fully off."Thresholding: After training, any gate value below $0.05$ is considered pruned.🚀 UsageInstall dependencies:Bashpip install -r requirements.txt
